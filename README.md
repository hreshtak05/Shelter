# БУНКЕР — веб-ведущий апокалипсиса

Веб-приложение для настольной игры **«Бункер: Поколение Альфа»**: драматично зачитывает
катастрофы под тревожную музыку, ведёт раунды и таймеры, объявляет изгнанных. Интерфейс
и контент — на русском.

## Что уже работает (Фаза 1)

- Экран настройки: список игроков (4–16), выбор режима (Базовый / История выживания).
- Сетка катастроф → клик запускает музыку и зачитывает описание.
- Игровой движок по **Таблице раундов** (правила, стр. 5): сам знает, в каком раунде
  голосование, сколько изгнаний и сколько мест в бункере — в зависимости от числа игроков.
- Круг открытия карт с таймером **30 сек** на игрока, переход по кругу.
- Таймер обсуждения (2 / 5 / произвольно минут), кнопки **+10с / +20с**, **Стоп**.
- Тик-так во время таймеров, драматичный «стингер» при изгнании, ducking музыки под голос.
- Голосование: отметка изгнанных и драматичное объявление имени.
- Финал: кто в бункере, кто изгнан.

> Звук в Фазе 1 **синтезируется** (Web Audio API), голос — встроенный в браузер
> (`speechSynthesis`, ru-RU). Файлы ассетов не нужны, всё звучит сразу.

## Запуск

Нужен любой статический сервер (из-за `fetch` к JSON):

```bash
cd bunker-app
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Структура

```
bunker-app/
  index.html        экраны
  styles.css        тёмная «бункерная» тема
  app.js            движок: состояние, экраны, таймеры, голосование
  audio.js          музыка/тик-так/стингер (Web Audio) + голос
  data/
    round-table.json  Таблица раундов (стр. 5) — голосования и места по числу игроков
    catastrophes.json ⚠ ВРЕМЕННЫЕ заглушки — заменить точным текстом с карт
    phrases.json      фиксированные реплики ведущего
  scripts/
    generate-audio.mjs  Фаза 4: генерация MP3-озвучки катастроф (ElevenLabs)
```

## Дальнейшие фазы

- **Фаза 3 — контент.** Заменить заглушки в `data/catastrophes.json` точным текстом ваших
  карт Катастроф (по фото). Поле `placeholder` убрать.
- **Фаза 4 — кинематографичный голос (Gemini).** Получить ключ
  (https://aistudio.google.com/apikey) и запустить:
  ```bash
  GEMINI_API_KEY=xxx node scripts/generate-audio-gemini.mjs
  # необязательно: VOICE_NAME=Charon (мужские: Charon, Fenrir, Orus, Algenib …)
  ```
  Скрипт создаст аудио в `audio/voice/` мужским драматичным голосом и пропишет его в
  `catastrophes.json`. Приложение начнёт играть готовые файлы вместо голоса браузера —
  драматично и без галлюцинаций. Ключ остаётся приватным (в публичный сайт не попадает).
  Имена изгнанных игроков по-прежнему озвучивает голос браузера (имена вводятся каждую игру).
  *(Вариант на ElevenLabs остаётся в `scripts/generate-audio.mjs`.)*

## Своя музыка

Положите свой трек как **`audio/music/theme.mp3`** — он будет фоном (зациклен, с
плавным затуханием под голос). Файла нет → просто тишина, без синтезированного «шума».
Имя/путь можно поменять в начале `audio.js` (`MUSIC_SRC`).
