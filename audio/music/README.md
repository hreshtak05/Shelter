# Фоновая музыка

Положите сюда свой трек и назовите его **`theme.mp3`** — приложение будет
проигрывать его зацикленно (loop) как фоновую музыку, с плавным затуханием
под голос. Кнопка «🔊 Музыка» в углу включает/выключает фон.

- Путь по умолчанию: `audio/music/theme.mp3`
- Поменять имя/путь можно в начале `audio.js` (`MUSIC_SRC`).
- Файла нет → просто тишина (без синтезированного «шума»).
